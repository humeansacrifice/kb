Welcome

Skrifa is a Knowledge Base theme for Jekyll. I learned Liquid and Jekyll, and built it in one drunk weekend, so it's not quite perfect.

I borrowed some code and stole piles of inspiration from Tom Johnson's Jekyll Documentation Theme. Skrifa is using Bootswatch's rad-ass Bootstrap Flatly Theme.

Notes

  • This theme is very much a work-in-progress, so some things are a bit weird, or may not work as expected -- pull requests are welcome.
  • This theme doesn't use posts. Add 'em if you need 'em.
  • Some of the CSS is hacky. It's minimal and limited to the TOC and some unordered list items. I don't understand CSS.
  • The search doesn't work well. I was too drunk to try to get it to work the way I wanted it to. It's on my roadmap.
